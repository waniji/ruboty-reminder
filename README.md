# Ruboty::Reminder

Ruboty handler to remind a task.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ruboty-reminder'
```

## Usage

```
@ruboty add task HH:MM <task>
@ruboty delete task <id>
@ruboty list tasks <id>
```

## Example

```
> @ruboty add task 07:00 Hi,kaihara!
Task 270 created.
```
